# Les Commandes PowerShell

#### **Les commandes PowerShell et Les commandes Linux**

**Powershell**   ---->  **ShellBash**  

 Copy-Item ---->  cp  
 Remove-Item ->  rm  
 Set-Location -->  cd  
 New-Item ----->  mkdir  
 help  ----------->  man  
 Get-History --->  history  
 Get-Alias ------>  alias  
 Get-Content -->  cat
